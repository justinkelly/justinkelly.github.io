--- layout: post title: Simple Invoices 2010.2 Beta 2 has been released published: true tags: - simple invoices --- Hi Guys,

Simple Invoices 2010.2 Beta 2 has been released.

You can grab the download from:
http://simpleinvoices.googlecode.com/files/simpleinvoices.2010.2.beta2.zip

This issue fixes a few issues from the beta 1 release include:
* Fixed tax total and grouping on invoice print issue
* Fixed issue with various 'manage' pages grid not working
* Fixed issue where blank screen presented if tmp dir not write-able
* refer http://code.google.com/p/simpleinvoices/wiki/done#2010.2_Beta_2
for full list

Documentation:
* Installing Simple Invoices: http://www.simpleinvoices.org/install

* Simple Invoices help: http://www.simpleinvoices.org/help
* Frequently Asked Questions: http://www.simpleinvoices.org/wiki/faqs
* Simple Invoices config file options: http://www.simpleinvoices.org/config

import the Simple_Invoices_Full.sql file for new dbs
