--- layout: post title: Simple Invoices 2010.2 Beta 3 released published: true tags: - simple invoices --- Hi Guys,

Simple Invoices 2010.2 Beta 3 has been released.

You can grab the download from:
http://simpleinvoices.googlecode.com/files/simpleinvoices.2010.2.beta3.zip

This issue fixes a few issues from the beta 1 release include:
* Added Inventory - refer http://simpleinvoices.org/wiki/inventory
* fixed add user no role problem
* fixed decimal place - missing zeros problem
* fixed cron ACL issue
* fixed expense extension/2010.2 issues

Documentation:
* Installing Simple Invoices: http://www.simpleinvoices.org/install
* Upgrading Simple Invoices: http://www.simpleinvoices.org/wiki/upgrade
* Simple Invoices help: http://www.simpleinvoices.org/help
* Frequently Asked Questions: http://www.simpleinvoices.org/wiki/faqs
* Simple Invoices config file options: http://www.simpleinvoices.org/config

Known issues:
* Not all of the issues with Simple Invoices 2010.2 beta 2 were
resolved with this release - refer
http://code.google.com/p/simpleinvoices/wiki/todo
* Automatic installer might not work 100% - best bet is to manually
import the Simple_Invoices_Full.sql file for new dbs
* French translation - the manage grids don't work

If you have any issues or queries please post in the forum:
* http://simpleinvoices.org/forum/discussion/1182/simple-invoices-20102-beta-3-...
