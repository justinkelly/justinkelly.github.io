what i was doing

{% highlight php %} echo $this->url(array('controller'='pages','action'='about')); {% endhighlight %}

the correct way for zend framework $this->url links

{% highlight php %} echo $this->url(array('controller'='pages','action'='about'),null,true); {% endhighlight %}
